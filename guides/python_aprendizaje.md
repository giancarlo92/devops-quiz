# Python – Guía Rápida

## Definición
- Lenguaje de propósito general con ecosistema amplio.

## Comandos básicos
- Entorno: `python -m venv .venv && source .venv/bin/activate`
- Instalación: `pip install -r requirements.txt`
- Tests: `pytest -q`

## Casos prácticos
- Scripting de automatización e integración con APIs.

## Estructura básica (app)
```
app/
  __init__.py
  main.py
requirements.txt
```

## Buenas prácticas
- Usar `venv`, tipado (`typing`) y formateo (black/flake8).
